# 📊 Financial Analysis of a Condominium | Data Analytics Project

## 📌 Project Overview
This project demonstrates an end-to-end **data analytics workflow** applied to condominium financial data.  
It covers **data extraction, transformation (ETL), exploratory analysis (EDA), and dashboarding**, enabling better financial visibility and decision-making.

The solution transforms raw bank statement data into actionable insights through a structured pipeline and an interactive dashboard.

---

## 🎯 Business Problem
Condominium financial management often lacks:
- Clear visibility of expenses and revenues  
- Standardized categorization of transactions  
- Data-driven insights for decision-making  

This project addresses these challenges by building a **data pipeline and analytical layer** to support financial control and transparency.

## 🔄 Data Pipeline (ETL)

### 🔹 Extract
- Import of raw financial data (bank statements)
- CSV/Excel ingestion

### 🔹 Transform
- Data cleaning (null handling, formatting)
- Standardization of dates and values
- Categorization of transactions (income vs expenses)
- Feature engineering (balance, grouped categories)

### 🔹 Load
- Structured dataset ready for analysis
- Data prepared for dashboard visualization
