# 📊 Análise Financeira de Condomínio | Projeto de Data Analytics

## 📌 Visão Geral do Projeto
Este projeto demonstra um fluxo completo de **análise de dados**, aplicado a dados financeiros de um condomínio.  
Abrange etapas de **ETL (Extração, Transformação e Carga)**, análise exploratória (EDA) e construção de dashboard interativo.

A solução transforma dados brutos de extratos bancários em **insights acionáveis**, apoiando a gestão financeira e a tomada de decisão.

---

## 🎯 Problema de Negócio
A gestão financeira de condomínios frequentemente enfrenta desafios como:
- Falta de visibilidade clara sobre receitas e despesas  
- Ausência de categorização padronizada  
- Dificuldade na geração de insights  

Este projeto resolve esses pontos ao construir uma **camada analítica estruturada** baseada em dados.

## 🔄 Pipeline de Dados (ETL)

### 🔹 Extração
- Importação de dados financeiros (extratos bancários)
- Leitura de arquivos CSV/Excel

### 🔹 Transformação
- Limpeza de dados (valores nulos, padronização)
- Tratamento de datas e valores
- Classificação de receitas e despesas
- Criação de métricas (saldo, categorias)

### 🔹 Carga
- Dados estruturados para análise
- Preparação para visualização no dashboard

---

## 📊 Principais Análises
- Evolução do saldo ao longo do tempo  
- Comparação entre receitas e despesas  
- Distribuição de gastos por categoria

- Identificação de padrões de consumo
